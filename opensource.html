Open Source

We contribute to the open-source community as a default. Tooling we build that is not client-specific is released publicly under the Resengi GitHub organization.

Published packages

  • hand_drawn_toolkit v0.4.0

    A lightweight Flutter package for hand-drawn UI: sketchy containers, dividers, notebook rows, status squares, text fields, tables, and interactive charts.

    Requires JavaScript to load the demo

  • page_turn_animation v0.1.4

    Realistic page turn / curl animation for transitioning between content using captured images. Note: this demo is not gesture controlled.

    Requires JavaScript to load the demo

  • recurrence_kit v0.1.1

    Standalone recurrence rule system with data models, computation engine, and picker UI widget.

    Requires JavaScript to load the demo

  • flip_calendar v0.1.2

    Customizable calendar widget with page-turn animations and swipe gesture navigation.

    Requires JavaScript to load the demo

More packages and tooling will be published as we grow. All open-source work is available on our GitHub organization and our pub.dev publisher page.

For clients

When we work on your project, any tooling we develop that is not specific to your codebase may be released as open source. This is part of how we operate, and it is discussed during contract negotiation. We can work out an exception in the contract if your codebase is sensitive, but openness is our starting point.

We find that work done with the intention of sharing tends to be higher quality, and we take pride in contributing useful tools back to the community we are a part of.

Resengi — from Research + Engineering. Every engagement is a chance to solve the problem in front of us and leave behind something useful for the wider world.
